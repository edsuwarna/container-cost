## Ikhtisar

Container Cost adalah **alat alokasi biaya container Docker** yang membantu menghitung dan memonitor berapa biaya setiap container di satu atau banyak VPS. Pake formula alokasi berbobot (weighted) berdasarkan pemakaian CPU dan RAM aktual.

### Untuk Siapa?

| Segmen | Use Case |
|--------|----------|
| **Indie Developer / Solo Dev** | Side project di 1-3 VPS, tau breakdown biaya |
| **DevOps Engineer** | Deploy multi-tenant, chargeback ke tim/klien |
| **Agen / Konsultan** | Kelola infra klien, billing transparan |
