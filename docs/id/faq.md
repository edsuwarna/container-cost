# Container Cost — FAQ ❓

**Q: Berapa banyak VPS yang bisa dikelola?**
A: Secara teori tidak terbatas, dependen sama resource central server.

**Q: Agent pake resource berapa?**
A: ~15MB RAM, 0.1% CPU. Sangat ringan.

**Q: Bisa pake provider cloud apa aja?**
A: Bisa, selama ada Docker dan koneksi ke central server.

**Q: Data history disimpen berapa lama?**
A: Selamanya (sampai dihapus). Bisa diatur retention policy.
