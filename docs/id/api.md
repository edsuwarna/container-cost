# Container Cost — Referensi API 🔌

## Endpoints

### GET /api/containers
Daftar semua container dengan biaya.

### GET /api/vps
Daftar semua VPS dengan ringkasan biaya.

### GET /api/agents
Daftar agent yang terhubung.

### POST /api/agents/register
Daftarkan agent baru.

### GET /api/costs/summary
Ringkasan biaya semua VPS + container.
