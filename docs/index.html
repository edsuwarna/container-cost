Agent Mode

Deploy Agent on Your VPS

311| 311|

Run this one-liner on any VPS with Docker installed:

312| 312|
313| 313|
$ curl -fsSL https://raw.githubusercontent.com/edsuwarna/container-cost/main/deploy/setup-agent.sh | bash -s -- \
   314|   314|  --server=http://YOUR_SERVER_IP:8080 \
   315|   315|  --api-key=dckr_xxx_generated_from_dashboard \
   316|   316|  --name="My VPS" \
   317|   317|  --price=200000 \
   318|   318|  --cpu=4 \
   319|   319|  --ram=8
