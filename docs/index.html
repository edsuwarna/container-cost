✨ Key Features

238|

What Container Cost brings to your infrastructure

239|
240|
241|
📊
242|

Multi-VPS Dashboard

243|

See total cost, per-container breakdown, and trends from all VPS in one central dashboard. Aggregated views across every agent.

244|
245|
246|
🤖
247|

Agent Mode

248|

Deploy a lightweight agent on each VPS. Agents calculate costs locally and push reports to the central server via a secure API.

249|
250|
251|
🧮
252|

Weighted Cost Formula

253|

Fair cost allocation based on actual resource usage: CPU 50%, RAM 40%, Storage 10%. Fully adjustable weights.

254|
255|
256|
🔐
257|

Role-based Auth

258|

Three roles — Admin, Engineer, Management — each with appropriate access levels. Session-based auth with secure cookies.

259|
260|

🖥️

VPS Management

263|

Add, edit, and delete VPS from the dashboard. Auto-generate API keys, monitor online/offline status, track last seen.

⚡ Quick Install

278|

Get started in under 5 minutes

279| 280|
281|
282|
1

Clone the Repository

285|

Clone the project and start the central server with PostgreSQL.

286|
287|
$ git clone https://github.com/edsuwarna/container-cost.git
   288|$ cd container-cost
   289|$ docker compose up -d
   290|[+] Running 3/3
   291| ✔ Network created
   292| ✔ Container postgres  Started
   293| ✔ Container container-cost  Started
   294|$ echo "Open http://localhost:8081"
   295|Open http://localhost:8081
296|
297|

Default login: admin / change-me

298|
299|
300|
301|
2
302|
303|

Add a VPS & Generate API Key

304|

Login to the dashboard, go to VPS menu, click Add VPS, and generate an API key for each VPS you want to monitor.

305|
306|
307|
308|
3

310|

Deploy Agent on Your VPS

311|

Run this one-liner on any VPS with Docker installed:

312|
313|
$ curl -fsSL https://raw.githubusercontent.com/edsuwarna/container-cost/main/deploy/setup-agent.sh | bash -s -- \
   314|  --server=http://YOUR_SERVER_IP:8080 \
   315|  --api-key=dckr_xxx_generated_from_dashboard \
   316|  --name="My VPS" \
   317|  --price=200000 \
   318|  --cpu=4 \
   319|  --ram=8
