Container Cost

Docker Container Cost Calculator dengan arsitektur Agent + Central Server. Pantau biaya semua container di seluruh VPS dari satu dashboard.

📂 View on GitHub 📖 Full Documentation 🐳 Docker Image

✨ Features

Apa yang bisa dilakukan Container Cost?

📊

Multi-VPS Dashboard

Lihat total cost, per-container breakdown, dan tren dari semua VPS dalam satu dashboard terpusat.

🤖

Agent Mode

Deploy agent ringan di tiap VPS. Agent ngitung cost masing-masing, push ke central server.

🧮

Weighted Cost Formula

Alokasi biaya berdasarkan resource aktual: CPU 50%, RAM 40%, Storage 10%.

🔐

Role-based Auth

Tiga role: Admin, Engineer, Management — beda akses sesuai kebutuhan.

🖥️

VPS Management

Tambah/hapus VPS dari dashboard, auto-generate API key, monitoring online/offline.

🚀

One-liner Deploy

Deploy agent di VPS baru dengan satu command curl ... | bash.

⚡ Quick Start

Deploy central server + agent dalam 5 menit

1

Clone & Start Central Server

Jalankan PostgreSQL dan Container Cost dengan satu perintah.

git clone https://github.com/edsuwarna/container-cost.git
cd container-cost
docker compose up -d
# Buka http://localhost:8081

Login: admin / change-me

2

Add VPS dari Dashboard

Login → VPS menu → Tambah VPS → dapatkan API Key.

3

Deploy Agent di VPS Lain

Satu command deploy agent di VPS mana pun yang punya Docker.

curl -fsSL https://raw.githubusercontent.com/edsuwarna/container-cost/\nmain/deploy/setup-agent.sh | bash -s -- \\\n  --server=http://CENTRAL_IP:8080 \\\n  --api-key=dckr_xxx \\\n  --name="Hetzner CX42" \\\n  --price=200000 \\\n  --cpu=4 \\\n  --ram=8
4

Lihat Hasilnya

Dashboard otomatis nampilin data semua VPS — cost breakdown, tren, dan status.
